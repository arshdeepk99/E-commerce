# 🛒 E-Commerce Website

A fully functional and dynamic **E-commerce web application** built using **Spring Boot**, **Spring MVC**, **Hibernate**, and **Thymeleaf**, with a responsive frontend using **HTML, CSS, and Bootstrap**. This project allows users to browse products, register/login, add items to the cart, and place orders.


---

## 🛠️ Tech Stack

### 🔹 Backend:
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL / H2 Database

### 🔹 Frontend:
- HTML5, CSS3
- Bootstrap 5
- Thymeleaf Template Engine
